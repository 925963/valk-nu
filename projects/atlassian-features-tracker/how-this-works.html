$ man features

This tracker follows features published in the Atlassian Cloud weekly release notes. Every week Atlassian lists changes to their cloud products; a scraper reads those pages, deduplicates each feature across the weeks it appears, and records how its status changes over time.

Status labels

TAGMEANING
SOONAnnounced, not yet rolling out.
NEWJust started rolling out in the latest tracked week.
ROLLINGGradually being deployed to sites.
COMPLETERollout finished — dropped off the weekly notes.

How deduplication works

A feature often appears across many consecutive weekly pages while it rolls out. The scraper:

  1. Detects each feature by a stable ID derived from its title and product.
  2. Records the first week it appeared as the announced date.
  3. Tracks status changes (e.g. coming soon → rolling out).
  4. Marks a feature COMPLETE once it disappears from several consecutive weeks.

Update schedule

The scraper runs weekly via GitHub Actions in the atlassian-features repository. This page is a static, terminal-styled front-end built from a snapshot of that data — re-import and rebuild to refresh it.
